### Integration notes for Curry
- **The variant-switching system (`SectionVariantDial`, `SectionVariantSlot`, `sectionVariants.tsx`) is prototype-only scaffolding.** In production, pick the winning variant per section and render it directly — no URL-param routing needed.
- **Product videos are large** (`slides-website.mp4` is 35 MB). For production, transcode to WebM + H.265, serve from a CDN with range requests, and lazy-load with `preload="none"` or intersection-gated playback.
- **Wheel carousel math** (V5D) uses hardcoded constants `R=1000`, `STEP=30` tuned for a 680px-high box at desktop. If the container height changes significantly, recalculate `R` so `R*sin(STEP°)` still exceeds card height.
- **`StepMedia` sequence intervals** default to 1800ms; the data model supports per-point `intervalMs` overrides. The `layers` media kind is stubbed but unimplemented — reserved for decomposed-layer animations if design requests them.
