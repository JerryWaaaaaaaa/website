- **Mono icons use CSS `mask-image` for tinting.** The `psuite-tab-icon--mono` class applies a `mask-image: var(--icon)` with `background-color: currentColor`. This requires all mono SVGs to have solid `fill="#000000"` — if you rebuild these as React components or sprite sheets, preserve that fill for the mask to work.
- **Third-party brand assets are checked in as static files.** Competitor logos (Google, Microsoft, Notion) and integration logos (Claude, OpenAI) live in `public/competitor-logos/` and `public/integration-logos/`. For production, confirm legal/brand approval and consider hosting on a CDN instead of bundling. The `notion.png` and integration PNGs are large raster files (11–549 KB).
- **PositioningMatrix uses `cqw` (container-query-width) units** for responsive label sizing inside `pm-stage` with `container-type: inline-size`. Ensure your production CSS baseline supports container queries (all modern browsers do, but verify against your browser matrix). The mobile fallback at `max-width: 639px` linearizes the chart into a vertical stack with the SVG axes hidden.
